## IROK 磁轴键盘常见关注点

### RT（快速触发）

RT 更适合关注急停、连续修正输入、触发逻辑灵敏度的用户，常见于 FPS 和动作类游戏讨论场景。

### 8K 回报率

8K 回报率更常见于偏竞技定位的键盘配置，主要意义在于更高的数据回传频率。对高刷显示器和高帧率游戏用户更值得关注。

### 配列选择

63 配列更适合想给鼠标留出更多空间、桌面较小或更偏游戏使用的用户。68 配列通常更适合希望兼顾日常便利性和游戏体验的用户。75 配列则更接近兼顾功能区与效率需求的方向。

### 有线 / 无线 / 三模

固定桌面、主机游戏和稳定连接需求通常更适合有线方案；如果更看重笔记本、多设备切换、宿舍或移动使用场景，通常更需要无线或三模能力。
