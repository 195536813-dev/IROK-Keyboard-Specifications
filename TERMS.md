## 核心性能术语

### RT

RT 通常可以理解为快速触发能力，常见于磁轴键盘。它更适合急停、反复横拉、连续修正输入这类对按键起落更敏感的游戏场景。

对普通用户来说，RT 不是“自动提升操作”的功能，而是让按键触发和复位逻辑更灵敏，更贴近竞技向输入习惯。

### RT 精度

RT 精度通常表示触发行程调节的细化程度，例如 `0.005mm`、`0.001mm` 这类标注。

这类数字更适合用来理解“可调节能力的上限”，而不是简单理解成数字越小，所有用户的体验就一定越强。实际体验还会受到驱动调节、游戏场景和个人习惯影响。

### 响应延迟

响应延迟通常用于描述输入信号从触发到传输的速度表现。对于竞技用户来说，延迟越低越容易成为卖点，但实际感知还取决于整套设备环境，例如显示器刷新率、游戏帧率和系统状态。

### 回报率

回报率表示键盘向电脑回传数据的频率。`8000Hz` 也常写作 `8K`，表示每秒进行 8000 次数据回传。

更高的回报率理论上能带来更快的输入反馈，但它通常更适合高刷显示器、高帧率游戏和本身对输入延迟更敏感的用户。

### 扫描率

扫描率可以理解为键盘检测按键信号的频率。常见写法包括 `16kHz 全键扫描`、`32kHz 全键扫描`、`256kHz 单键扫描` 等。

对用户来说，扫描率和回报率一样，都是竞技键盘常见的性能参数。实际选购时，更值得关注的是稳定性、误触控制和整体体验，而不只是参数本身。

## 结构与用料术语

### 配列

配列指键盘的键位布局和尺寸形式，例如 `63 配列`、`68 配列`、`75 配列`。

- `63 配列`：更紧凑，更适合给鼠标让位置
- `68 配列`：通常比 63 配列更适合兼顾日常使用
- `75 配列`：功能区更完整，但桌面占用通常更大

### 定位板

定位板是键盘内部用于固定轴体、影响手感和结构稳定性的部件。常见材质包括铝合金、PC 等，不同材质会影响整体敲击反馈。

### 内部填充

内部填充通常用于控制空腔音、杂音和整体敲击感，例如 Poron 夹心棉、硅胶底垫、PU 泡棉等。它不一定直接决定性能，但会明显影响手感和声音风格。

### Gasket 结构

Gasket 是一种常见键盘结构方案，常用于改善敲击回弹和减少生硬感。对用户来说，它通常代表更柔和一点的敲击反馈和更完整一点的结构体验。

### PCB 防水 / IP54

PCB 防水表示键盘电路板具备一定基础防护能力。`IP54` 一类标注通常更偏向基础泼溅防护，不建议理解成可直接大量接触液体。

## 竞技功能缩写

### SOCD

SOCD 的常见英文写法是 `Simultaneous Opposing Cardinal Directions`，可理解为对相反方向输入的冲突处理。

它的作用是在两个相反方向按键同时被按下时，按预设逻辑输出正确结果，避免输入冲突。在部分竞技游戏场景里，这类功能更容易被关注，尤其是涉及急停、方向修正和快速反向输入时。

### 按键宏

按键宏的英文通常可对应 `Macro`。它表示可以通过驱动录制一系列按键操作，再将这些操作绑定到一个按键上，从而实现自动化重复功能。

这类功能适合快捷指令、自定义组合操作等场景，但是否适合使用，还需要结合具体场景、游戏平台规则和赛事规范判断。

### TGL

根据当前补充资料，TGL 的全称为 `Tangential Guide Linear`，可暂按“切换开关”理解。

它的核心逻辑是按键状态持久化，每次点击都可以切换按键的持续触发状态，例如在开启和关闭之间切换。更适合需要持续动作切换的场景。

### MT

根据当前补充资料，MT 的全称为 `Magnetic Type`，可暂按“按住 / 单击双功能”理解。

它支持通过单击和长按触发两种独立功能。对用户来说，更像是一颗键承担两种操作逻辑，适合需要主功能和副功能共用同一键位的场景。

### MPT

根据当前补充资料，MPT 的全称为 `Magnetic Position Tracking`，可理解为“多段触发”。

它支持在一次按压过程中，按不同按压深度分阶段触发不同指令。对用户来说，这意味着一个键可以承担多段操作逻辑，更适合对输入层次有要求的使用场景。

### END

根据当前补充资料，END 的全称为 `End Point Detection`，可理解为“松开触发”。

它支持在按下和松开两个阶段分别触发不同指令，例如按下触发一个键，松开时再触发另一个键。更适合需要利用回弹动作完成额外输入的场景。

### 竞技功能速览

| 功能 | 当前定义 | 典型理解 |
| :--- | :--- | :--- |
| SOCD | 相反方向输入冲突处理 | 更适合急停、反向输入、方向修正 |
| TGL | 切换开关 | 适合持续动作的开启 / 关闭切换 |
| MT | 单击 / 长按双功能 | 一颗键承担两种操作 |
| MPT | 多段触发 | 一次按压按深度触发多段指令 |
| END | 松开触发 | 按下和松开分别触发不同指令 |
| Macro | 按键宏 | 录制并复用组合操作 |

## 常见选购理解

### 63 配列和 68 配列怎么选

- 更看重鼠标空间和紧凑感：优先看 63 配列
- 更看重方向键和日常便利性：优先看 68 配列

### 8K 是不是一定比普通键盘强很多

理论上更高的回报率代表更快的数据回传，但并不是所有用户都能明显感知差异。它更适合竞技环境，而不是所有使用场景下都一定有同等提升。

### 磁轴键盘适合谁

更适合对快速触发、竞技输入、RT 调节这类特性有明确兴趣的用户。如果更在意完整办公体验、安静手感或无线多场景使用，则需要结合其他型号一起判断。

### 这些功能定义会不会变化

会。不同品牌、不同驱动版本、甚至同品牌不同型号之间，功能名称和触发逻辑都可能略有差异。因此在做精确定义、对外传播或售后说明时，最好结合当期驱动页面和产品说明书一起确认。
