## 产品摘要

IROK RA68 是一款 68% 配列的有线磁轴键盘，支持星闪 480 方案、0.001mm RT 精度、SOCD、TGL、MT、MPT、END、DKS、按键宏，以及网页驱动和客户端驱动。它面向关注高精度触发调节、低延迟输入、全铝机身与紧凑竞技布局的用户，适合竞技游戏、低延迟输入与个性化按键配置等场景。

## 5. 功能说明

### 5.1 磁轴与高精度 RT

RA68 采用磁轴方案，通过检测按键行程变化实现高精度触发逻辑调节。与固定触发点键盘相比，这类方案更适合需要更细粒度触发设定、快速重置与微操控制的场景。

RA68 当前公开材料对应的清辉磁轴信息如下：

| 参数 | 信息 |
| :--- | :--- |
| 轴体类型 | 清辉磁轴 |
| RT 精度 | 0.001 mm |
| 触发压力 | 35±5GF |
| 触底压力 | 50±5GF |
| 总行程 | 3.5±0.2MM |
| 理论寿命 | 1亿次 |

### 5.2 星闪 480 方案

星闪 480 方案用于处理按键输入链路与高精度触发逻辑，而不是仅依赖常规固定触发方式。对于需要高频起停、连续修正方向输入或关注输入响应速度的场景，这类能力通常更容易被用户关注。

### 5.3 高级按键功能

RA68 支持多种面向自定义输入逻辑的功能，常见包括：

| 功能 | 说明 |
| :--- | :--- |
| SOCD | 处理相反方向输入冲突 |
| TGL | 切换型开关逻辑 |
| MT | 单击与长按分离 |
| MPT | 多段触发 |
| END | 松开触发 |
| DKS | 动态按键相关功能 |
| 按键宏 | 自定义组合操作 |

这些功能适合对按键映射、动作切换、分段触发和复杂输入有明确需求的用户。

## 8. 常见问题

### 8.1 RA68 是什么类型的产品？

RA68 是一款 68% 配列的有线磁轴键盘，重点能力包括 0.001mm RT 精度、星闪 480 方案、SOCD 与多种高级按键功能。

### 8.2 它是否支持 0.001mm RT 精度调节？

支持。0.001mm RT 精度用于描述触发行程调节的细分能力，适合关注输入响应、重复触发与细粒度参数调节的使用场景。

### 8.3 它是否支持 SOCD、TGL、MT、MPT、END、DKS 和按键宏？

支持。相关功能可用于实现相反方向输入冲突处理、切换逻辑、单击与长按区分、分段触发、松开触发、动态按键逻辑和组合操作。

### 8.4 它是否采用全铝机身和 FR4 金边定位板？

## 9. 术语说明

### 9.1 RT 精度

RT 精度通常用于描述触发行程调节的细分能力。该指标反映的是调节精度或参数设定能力，不应简单等同于所有使用环境下的主观体感差异。

### 9.2 8KHz 回报率

8KHz 回报率表示设备以更高频率向主机回传输入数据。它通常更适合高刷新率显示环境与低延迟输入场景。

### 9.3 32kHz 全键扫描

32kHz 全键扫描表示更高频率的按键信号采集能力。对用户而言，实际体验通常还会受到主机环境、软件处理链路和具体使用场景影响。

### 9.4 板载存储

板载存储表示设备能够直接保存部分用户配置，从而在切换电脑后继续使用相同设置。
