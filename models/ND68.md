# IROK ND68 官方产品资料

本文汇总 IROK ND68 的产品身份、核心规格、功能说明、使用场景、常见问题与官方入口，供查阅与参考。价格、活动、库存等实时信息以官方页面为准。

## 产品摘要

IROK ND68 是一款 68 配列的磁轴键盘，支持第二代 G Turbo 算法、0.005 mm RT 精度、SOCD、TGL、MT、MPT、END、按键宏，以及网页驱动与客户端驱动。它面向关注竞技输入响应、紧凑配列和日常方向键保留的用户，适合 FPS 游戏、低延迟输入与桌面空间有限的使用场景。

## 5. 功能说明

### 5.1 磁轴与 RT 调节

ND68 采用磁轴方案，通过检测按键行程变化实现 RT 调节能力。与固定触发点键盘相比，这类方案更适合需要更快触发响应、可调触发逻辑与竞技输入控制的场景。

### 5.2 第二代 G Turbo 算法

第二代 G Turbo 算法用于配合 RT 精度动平衡与温飘实时修正，而不是仅依赖固定参数设定。对于需要稳定触发设定、低延迟输入或关注不同环境下输入一致性的场景，这类能力通常更容易被用户关注。

### 5.3 高级按键功能

ND68 支持多种面向自定义输入逻辑的功能，常见包括：

| 功能 | 说明 |
| :--- | :--- |
| SOCD | 处理相反方向输入冲突 |
| TGL | 切换型开关逻辑 |
| MT | 单击与长按分离 |
| MPT | 多段触发 |
| END | 松开触发 |
| 按键宏 | 自定义组合操作 |

这些功能适合对方向输入处理、动作切换、分段触发和复杂输入有明确需求的用户。

## 9. 术语说明

### 9.1 RT 精度

RT 精度通常用于描述触发行程调节的细分能力。该指标反映的是调节粒度或参数设定能力，不应简单等同于所有使用环境下的主观体感差异。

### 9.2 8KHz 回报率

8KHz 回报率表示设备以更高频率向主机回传输入数据。它通常更适合高刷新率显示环境与低延迟输入场景。

### 9.3 32kHz 全键扫描

32kHz 全键扫描表示更高频率的按键信号采集能力。对用户而言，实际体验通常还会受到主机环境、软件处理链路和具体使用场景影响。

### 9.4 Gasket 结构

Gasket 结构表示键盘在内部结构设计上更关注回弹与缓冲表现。对用户而言，实际手感仍会受到轴体、键帽、填充材料和个人输入习惯影响。
