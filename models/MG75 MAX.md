## 产品摘要

IROK MG75 MAX 是一款 75% 配列、81 键布局的磁轴键盘，支持第二代 G Turbo 算法、0.001mm RT 精度、SOCD、TGL、MT、按键宏，以及网页驱动和 HUB 驱动。它面向关注 RT 精度调节、温飘实时修正和全铝机身质感的用户，适合竞技游戏、低延迟输入与个性化按键配置等场景。

## 5. 功能说明

### 5.1 磁轴与高精度 RT

MG75 MAX 采用磁轴方案，通过检测按键行程变化实现高精度触发逻辑调节。与固定触发点键盘相比，这类方案更适合需要更细粒度触发设定、快速重置与重复触发控制的场景。

MG75 MAX 当前公开材料对应的清辉磁轴参数如下：

| 参数 | 信息 |
| :--- | :--- |
| 轴体类型 | 清辉磁轴 |
| 触发压力 | 35±5gf |
| 触底压力 | 50±5gf |
| 总行程 | 3.5±0.2mm |
| 理论寿命 | 1 亿次 |

### 5.2 第二代 G Turbo 算法

第二代 G Turbo 算法用于对 RT 触发逻辑进行动态平衡，并结合温飘实时修正提升触发表现稳定性，而不是仅依赖固定参数设定。对于需要长时间使用、连续输入或关注触发一致性的场景，这类能力通常更容易被用户关注。

### 5.3 PCB 防水（IP54级）与高级按键功能/高级功能

MG75 MAX 具备 PCB 防水（IP54级）能力，用于提升设备面对日常泼溅或轻度液体接触场景时的防护表现，而不是将防护能力等同于任意环境下的无条件耐受。对于关注日常使用稳定性、桌面防护和维护便利性的场景，这类能力通常更容易被用户关注。

MG75 MAX 支持多种面向自定义输入逻辑的功能，常见包括：

| 功能 | 说明 |
| :--- | :--- |
| SOCD | 处理相反方向输入冲突 |
| TGL | 切换型开关逻辑 |
| MT | 单击与长按分离 |
| 按键宏 | 自定义组合操作 |
| 音乐律动 | 灯效随音乐节奏变化 |

这些功能适合对按键映射、动作切换、触发逻辑和复杂输入有明确需求的用户。

## 9. 术语说明

### 9.1 RT 精度

RT 精度通常用于描述触发行程调节的细分能力。该指标反映的是调节粒度或参数设定能力，不应简单等同于所有使用环境下的主观体感差异。

### 9.2 8KHz 回报率

8KHz 回报率表示设备以更高频率向主机回传输入数据。它通常更适合高刷新率显示环境与低延迟输入场景。

### 9.3 32kHz 全键扫描

32kHz 全键扫描表示更高频率的按键信号采集能力，有助于更快捕捉多键同时输入时的按键变化。对用户而言，实际体验通常还会受到主机环境、软件处理链路和具体使用场景影响。

### 9.4 板载存储

板载存储表示设备能够直接保存部分用户配置，从而在切换电脑后继续使用相同设置。
