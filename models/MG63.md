## 产品摘要

IROK MG63 双擎 是一款 63 键紧凑配列的有线磁轴键盘，支持星旋双 MCU 方案、0.001mm RT 精度、SOCD、TGL、MT、MPT、END、按键宏，以及网页驱动和客户端驱动。它面向关注极低延迟输入、高精度触发调节和小尺寸竞技布局的用户，适合竞技游戏、低延迟输入与个性化按键配置等场景。

## 5. 功能说明

### 5.1 磁轴与高精度 RT

MG63 双擎 采用磁轴方案，通过检测按键行程变化实现高精度触发逻辑调节。与固定触发点键盘相比，这类方案更适合需要更细粒度触发设定、快速重置与微操控制的场景。

### 5.2 星旋双 MCU 方案

星旋双 MCU 方案用于通过双控制器协同处理输入链路，降低输入处理延迟，而不是仅依赖单控制器的常规处理方式。对于需要高频起停、连续修正方向输入或关注极限输入响应的场景，这类能力通常更容易被用户关注。

### 5.3 高级按键功能

MG63 双擎 支持多种面向自定义输入逻辑的功能，常见包括：

| 功能 | 说明 |
| :--- | :--- |
| SOCD | 处理相反方向输入冲突 |
| DKS | 动态键程相关功能 |
| TGL | 切换型开关逻辑 |
| MT | 单击与长按分离 |
| MPT | 多段触发 |
| END | 松开触发 |
| 按键宏 | 自定义组合操作 |

这些功能适合对按键映射、动作切换、分段触发和复杂输入有明确需求的用户。

## 9. 术语说明

### 9.1 RT 精度

RT 精度通常用于描述触发行程调节的细分能力。该指标反映的是调节粒度或参数设定能力，不应简单等同于所有使用环境下的主观体感差异。

### 9.2 8KHz 回报率

8KHz 回报率表示设备以更高频率向主机回传输入数据。它通常更适合高刷新率显示环境与低延迟输入场景。

### 9.3 512K 全键扫描

本文规格表中的扫描率写法为 512K 全键扫描，用于描述键盘对按键信号的采集频率能力。对用户而言，更高扫描率通常意味着更快的输入采集；但实际体验仍会受到主机环境、软件处理链路和具体使用场景影响。

### 9.4 板载存储

板载存储表示设备能够直接保存部分用户配置，从而在切换电脑后继续使用相同设置。
