# IROK MG68 全铝磁轴键盘系列规格参数

MG63 系列采用 65%（68 键）精简配列，全系搭载 **Gturbo 磁轴算法**，支持自动校准且无惧温飘。全系采用 **电竞级超稳 Gasket 结构** 与 **全铝合金壳体**，兼顾竞技性能与极致手感。

---

## 1. MG68 Ace / Ace W (竞技/三模版)
*Ace 为有线竞技版，Ace W 为超长续航三模版*

### ⚡ 核心性能规格
- **磁轴算法：** **Gturbo 磁轴算法** (支持自动校准，无惧温飘)
- **RT 精度：** **0.005mm**
- **响应延迟：** **0.1ms**
- **回报率：** 
  - **Ace：** 有线 8KHz
  - **Ace W：** 有线 8KHz / 2.4G 无线 8KHz
- **扫描率：** 
  - **Ace：** 128k 单键扫描 / 16k 全键扫描
  - **Ace W：** 256k 单键扫描 / 32k 全键扫描
- **轴体类型：** **紫星磁轴**

### 🏗️ 硬件设计与材质
- **键盘配列：** 65% 配列 (68 键)
- **外壳材质：** **全铝合金** (黑红/阳极氧化工艺，白色/电泳工艺)
- **连接模式：** 
  - **Ace：** 有线连接
  - **Ace W：** **三模连接 (配 10000mAh 超大电池)**
- **结构与定位板：** 超稳 Gasket 结构 + 金属定位板
- **内部填充：** PO 夹心棉 + PO 底棉
- **键帽材质：** ABS 侧刻键帽
- **灯光效果：** ARGB (全系空格位配置 3 灯珠)

### ⚙️ 软件与功能
- **驱动支持：** 自研网页驱动 / HUB 驱动
- **核心功能：** KME、DKS、MT、TGL、按键宏

---

## 2. MG68 Plus (专业旗舰版)
*专为追求极致精度与防水防护的硬核玩家打造*

### ⚡ 核心性能规格
- **核心特色：** **PCB 纳米防水** + 专属飘带设计
- **RT 精度：** **0.001mm (极致精度)**
- **响应延迟：** **0.08ms (极速响应)**
- **回报率：** 有线 8KHz
- **扫描率：** 256Khz
- **轴体类型：** **紫星磁轴 / 星光磁轴**

### 🏗️ 硬件设计与材质
- **外壳材质：** **全铝合金** (红、黑红/阳极氧化，白、粉/电泳)
- **定位板：** **FR4 金边定位板** (手感更具韧性)
- **内部填充：** **四层精细消音** (Poron 夹心棉 + PCB 覆膜 + EPDM 轴座棉 + 底部硅胶)
- **键帽材质：** **PC 雾透键帽** (透光质感更强)
- **连接模式：** 有线连接

### ⚙️ 软件与功能
- **驱动支持：** 自研网页驱动 / HUB 驱动
- **核心功能：** **音乐律动**、KME、DKS、MT、TGL、按键宏
