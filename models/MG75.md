# IROK MG75 全铝磁轴键盘系列规格参数

MG75 系列采用 75% 配列（81 键），是兼顾办公与竞技的黄金尺寸。全系采用 **CNC 全铝合金壳体** 与 **快拆设计**，并具备 **PCB 纳米防水** 能力。通过电竞级 Gasket 结构，为磁轴键盘提供更优越的按压手感与声性能。

---

## 1. MG75 Max (算法均衡版)
*搭载第二代 G Turbo 算法，主打温飘实时修正与动态平衡*

### ⚡ 核心性能规格
- **算法方案：** **第二代 G Turbo 算法** (支持 RT 精度动平衡、温飘实时修正)
- **RT 精度：** **0.001mm** (步进 0.001mm)
- **响应延迟：** **0.08ms**
- **回报率：** 8000Hz
- **扫描率：** 256Khz 单键扫描 / 32Khz 全键扫描
- **轴体类型：** **清辉磁轴**

### 🏗️ 硬件设计与材质
- **外壳材质：** **CNC 全铝合金** (阳极银/黑/红，电泳白/粉)
- **结构设计：** **快拆设计** + 电竞级 Gasket 结构
- **定位板：** 铝合金定位板
- **内部填充：** PO 夹心棉 + PCB 覆膜 + EPDM 底棉
- **键帽材质：** PC 雾透键帽
- **核心防护：** **PCB 防水**

### ⚙️ 软件与功能
- **驱动支持：** 网页驱动 / HUB 驱动
- **核心功能：** KME、音乐律动、TGL、MT、按键宏

---

## 2. MG75 V2 (旗舰双擎版)
*搭载星旋双 MCU 方案，极致 0.07ms 延迟与 IP67 级防护*

### ⚡ 核心性能规格
- **核心方案：** **星旋方案 (StarSpin)**，**双 MCU 全链路直连**
- **RT 精度：** **全键全段 0.001mm**
- **响应延迟：** **0.07ms (极速响应)**
- **回报率：** 8000Hz
- **扫描率：** 512K 全键扫描 / 147500K 单键扫描
- **轴体类型：** **清辉磁轴 PRO**

### 🏗️ 硬件设计与材质
- **核心特色：** **IP67 级 PCB 防水** + **锻波工艺背板**
- **外壳材质：** **CNC 全铝合金** (阳极银/黑/红)
- **结构设计：** **快拆设计** + 电竞级 Gasket 结构
- **内部填充：** PO 夹心棉 + PCB 覆膜 + EPDM 底棉 + **PET 垫片** (极致消音)
- **键帽材质：** PC 雾透键帽

### ⚙️ 软件与功能
- **核心竞技功能：** **SOCD (急停)**、DKS、MPT、END、TGL、MT
- **驱动支持：** 网页驱动 / HUB 驱动
- **附加特性：** 音乐律动灯效、全键自定义宏

---

## 🚀 MG75 系列核心卖点

- **75% 黄金配列：** 81 键位保留了完整的 F 区与独立方向键，在保证桌面鼠标空间的同时，兼顾了日常使用的便利性。
- **快拆结构设计：** 方便发烧友玩家进行内部维护或更换填充材料，体现了极高的 DIY 友好度。
- **顶级防护性能：** 全系配备防水涂层，MG75 V2 更达到了 IP67 级别，有效防护饮料泼溅带来的损坏。
- **双方案可选：** 追求算法稳定与实时修正温飘选择 **Max 版**；追求极致硬件响应与双 MCU 性能选择 **V2 版**。
