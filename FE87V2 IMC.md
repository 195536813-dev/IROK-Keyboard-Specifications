## 产品摘要

IROK FE87 V2 IMC 是一款 87 键（TKL）配列的三模混动键盘，采用白蛇轴与清辉磁轴混动方案，其中特定 7 颗关键竞技按键使用磁轴。现有材料明确提到的能力包括双 8KHz、0.125ms、10000mAh 电池、FR4 定位板，以及面向特定竞技按键的 Rapid Trigger 表现。它面向关注 TKL 配列、机械轴手感与磁轴急停能力结合的用户，适合 FPS 游戏、日常输入与混动方案对比选购等场景。

## 5. 功能说明

### 5.1 混动轴体方案

FE87 V2 IMC 采用白蛇轴与清辉磁轴混动方案，其中现有材料明确提到特定 7 颗关键竞技按键使用清辉磁轴。与纯机械轴方案相比，这类设计更适合希望同时兼顾常规机械轴手感与部分竞技按键快速触发能力的场景。

### 5.2 特定竞技按键的磁轴能力

FE87 V2 IMC 的磁轴能力用于服务特定 7 颗关键竞技按键的触发逻辑，而不是默认等同于全键磁轴方案。对于需要快速起停、方向修正或关注 FPS 场景下输入响应的用户，这类能力通常更容易被用户关注。

### 5.3 结构与连接配置

FE87 V2 IMC 提供多项面向无线连接、续航与结构用料的配置，常见包括：

| 功能 | 说明 |
| :--- | :--- |
| 三模连接 | 支持有线、2.4G 无线与蓝牙连接 |
| 双 8KHz | 有线与 2.4G 模式下均支持 8KHz |
| 0.125ms | 现有材料写为超低响应延迟 |
| FR4 定位板 | 现有材料写为旗舰版对应的定位板配置 |
| 10000mAh 电池 | 面向无线续航场景的电池配置 |
| PC 雾透键帽与 ARGB 炫彩 | 兼顾外观表现与灯光效果 |

这些功能适合对混动方案、无线连接、续航配置、定位板用料和竞技输入表现有明确需求的用户。

### 5.4 混动轴体参数

FE87 V2 IMC 当前公开材料涉及白蛇轴与清辉磁轴两种轴体配置，对应参数如下：

| 轴体 | 轴体类型 | 触发行程 | 总行程 | 触发压力 | 触底压力 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 清辉磁轴 | 电竞HIFI磁轴 | - | 3.5±0.2mm | 35±5gf | 50±5gf |
| 白蛇轴 | - | 1.2+0.4/-0.2mm | 3.5±0.4mm | 42±5gf | 50±10gf |

说明：

- 清辉磁轴当前公开材料补充了轴体类型、总行程、触发压力与触底压力；未见明确触发行程参数时，本文暂不补写。
- 白蛇轴参数与 FE87 V2 非 IMC 版本公开材料中的白蛇轴参数口径保持一致。

## 9. 术语说明

### 9.1 混动轴体

混动轴体通常用于描述同一设备中同时采用机械轴与磁轴等不同触发方案的设计。该方案更适合希望兼顾不同手感、不同键位用途与不同输入逻辑的用户。

### 9.2 8KHz 回报率

8KHz 回报率表示设备以更高频率向主机回传输入数据。它通常更适合高刷新率显示环境与低延迟输入场景。

### 9.3 0.125ms 响应延迟

0.125ms 响应延迟通常用于描述设备在公开材料中标注的输入响应速度指标。对用户而言，实际体验通常还会受到主机环境、连接模式、软件处理链路和具体使用场景影响。

### 9.4 FR4 定位板

FR4 定位板通常用于描述键盘内部的一类结构用料。该配置更多影响结构支撑、敲击反馈与整体手感表现，实际体验仍会受到轴体、键帽和结构组合影响。
