## 产品摘要

IROK MG68 PLUS 是一款 65% 配列、68 键布局的有线磁轴键盘，支持 0.001mm RT 精度、PCB 防水、SOCD、DKS、MT、TGL、按键宏，以及网页驱动和客户端驱动。它面向关注高精度触发、全铝合金机身和紧凑桌面布局的用户，适合竞技游戏、低延迟输入与个性化按键配置等场景。

## 5. 功能说明

### 5.1 磁轴与高精度 RT

MG68 PLUS 采用磁轴方案，通过检测按键行程变化实现高精度触发逻辑调节。与固定触发点键盘相比，这类方案更适合需要更细粒度触发设定、快速重置与微操控制的场景。

### 5.2 PCB 防水

PCB 防水用于提升设备面对日常泼溅或潮湿环境时的防护能力，而不是仅依赖常规裸露电路板结构。对于关注长期使用稳定性、桌面防护和日常维护便利性的场景，这类能力通常更容易被用户关注。

### 5.3 高级按键功能

MG68 PLUS 支持多种面向自定义输入逻辑的功能，常见包括：

| 功能 | 说明 |
| :--- | :--- |
| SOCD | 处理相反方向输入冲突 |
| DKS | 动态键程相关功能 |
| MT | 单击与长按分离 |
| TGL | 切换型开关逻辑 |
| 按键宏 | 自定义组合操作 |
| 音乐律动 | 灯效随音乐节奏变化 |

这些功能适合对按键映射、动作切换、键程逻辑和复杂输入有明确需求的用户。

## 9. 术语说明

### 9.1 RT 精度

RT 精度通常用于描述触发行程调节的细分能力。该指标反映的是调节粒度或参数设定能力，不应简单等同于所有使用环境下的主观体感差异。

### 9.2 8KHz 回报率

8KHz 回报率表示设备以更高频率向主机回传输入数据。它通常更适合高刷新率显示环境与低延迟输入场景。

### 9.3 256kHz 单键扫描

256kHz 单键扫描表示更高频率的单键按键信号采集能力。对用户而言，实际体验通常还会受到主机环境、软件处理链路和具体使用场景影响。

板载存储表示设备能够直接保存部分用户配置，从而在切换电脑后继续使用相同设置。
